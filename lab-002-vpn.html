Lab 002 · VPN PC → VPS

Objetivo: Túnel WireGuard punto a punto entre tu PC y un VPS con IP pública para evitar restricciones de CGNAT.

1. Topología

  • • VPS: Debian 13, IP pública (ej: 203.0.113.10).
  • • PC: Tu laptop (Debian/Arch) tras router/CGNAT.

2. Preparar VPS (Servidor)

# Instalar y generar claves
apt update && apt install -y wireguard
cd /etc/wireguard
umask 077
wg genkey | tee server.key | wg pubkey > server.pub

Archivo /etc/wireguard/wg0.conf:

[Interface]
Address = 10.10.10.1/24
ListenPort = 51820
PrivateKey = <contenido de server.key>
PostUp   = sysctl -w net.ipv4.ip_forward=1
PostDown = sysctl -w net.ipv4.ip_forward=0

Iniciar servicio:

systemctl enable wg-quick@wg0
systemctl start  wg-quick@wg0

3. Preparar Cliente (PC)

# Instalar y claves
sudo apt install wireguard # o pacman -S wireguard-tools
wg genkey | tee client.key | wg pubkey > client.pub

4. Configurar Cliente

Archivo /etc/wireguard/wg0.conf en tu PC:

[Interface]
Address = 10.10.10.2/32
PrivateKey = <client.key>

[Peer]
PublicKey = <server.pub>
Endpoint  = 203.0.113.10:51820
AllowedIPs = 10.10.10.0/24
PersistentKeepalive = 25

5. Conectar

sudo wg-quick up wg0
ping 10.10.10.1
