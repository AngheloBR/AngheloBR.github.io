Arch / CachyOS Configs

Documentación del entorno: paquetes base, terminal, zsh, powerlevel10k, y herramientas.

1. Paquetes base recomendados

# Ejemplo (ajusta a tu gusto)
sudo pacman -S --needed \
  base-devel git zsh neovim wget curl htop fastfetch \
  net-tools inetutils openssh

2. Shell y prompt

Cambiar shell por defecto a zsh:

chsh -s /usr/bin/zsh <tu_usuario>

Siguiente paso: instalar Oh My Zsh y powerlevel10k.
