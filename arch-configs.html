Arch / CachyOS Configs

Documentación de mi entorno de escritorio personal: paquetes base, shell, terminal y configuraciones. En progreso y creciendo con el tiempo.

01 Paquetes base recomendados

Conjunto mínimo de herramientas que instalo en cualquier sistema Arch/CachyOS fresco:

sudo pacman -S --needed \
  base-devel git zsh neovim wget curl \
  htop btop fastfetch unzip \
  net-tools inetutils openssh \
  man-db man-pages
ℹ

En CachyOS muchos de estos ya vienen preinstalados. Verifica con pacman -Q <paquete> antes de reinstalar.

02 Shell: Zsh + Oh My Zsh

# Cambiar shell por defecto
chsh -s /usr/bin/zsh

# Instalar Oh My Zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Plugins útiles

# zsh-autosuggestions
git clone https://github.com/zsh-users/zsh-autosuggestions \
  ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

# zsh-syntax-highlighting
git clone https://github.com/zsh-users/zsh-syntax-highlighting \
  ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

En ~/.zshrc:

plugins=(git zsh-autosuggestions zsh-syntax-highlighting z)

03 Alias útiles

# Añade a ~/.zshrc o ~/.bashrc

# Pacman / Paru
alias pac='sudo pacman -S'
alias pacr='sudo pacman -Rns'
alias pacu='sudo pacman -Syu'
alias pacs='pacman -Ss'
alias paci='pacman -Qi'

# Sistema
alias ll='ls -lah --color=auto'
alias cls='clear'
alias ip='ip --color=auto'

# Git
alias gs='git status'
alias gc='git commit -m'
alias gp='git push'
alias gl='git log --oneline --graph'
