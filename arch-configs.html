Configs base para Arch / CachyOS

Página para ir documentando tu entorno Arch/CachyOS: paquetes base, terminal, zsh, powerlevel10k, herramientas de hacking, etc.

1. Paquetes base recomendados

# Ejemplo (ajusta a tu gusto) sudo pacman -S --needed \ base-devel git zsh neovim wget curl htop fastfetch \ net-tools inetutils openssh

2. Shell y prompt

Cambiar la shell por defecto a zsh:

chsh -s /usr/bin/zsh <tu_usuario>

Ejemplo de siguiente paso: instalar Oh My Zsh y powerlevel10k (manual o AUR) y luego pegar aquí tus comandos reales cuando los tengas claros.
