Lab 004 · NAS casero Debian

Montaje de un NAS sencillo usando Samba para compartir archivos en tu red local.

1. Almacenamiento

# Crear sistema de archivos (Borra datos!)
sudo mkfs.ext4 /dev/sdb

# Montar
sudo mkdir -p /srv/nas/datos
sudo mount /dev/sdb /srv/nas/datos

2. Instalar Samba

sudo apt update && sudo apt install -y samba
sudo useradd -m nasuser
sudo passwd nasuser
sudo smbpasswd -a nasuser

3. Configurar Compartido

Permisos de carpeta:

sudo chown -R nasuser:nasuser /srv/nas/datos
sudo chmod -R 750 /srv/nas/datos

Añadir a /etc/samba/smb.conf:

[nas-datos]
   path = /srv/nas/datos
   browseable = yes
   read only = no
   valid users = nasuser
   create mask = 0640
   directory mask = 0750
sudo systemctl restart smbd

4. Acceso

Windows:

\\IP_DEL_NAS\nas-datos

Linux:

sudo mount -t cifs //IP/nas-datos /mnt -o username=nasuser
